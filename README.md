# Memory-Lane 🧠

An AI-powered facial memory assistant built with the MERN stack.

### Key Features
* **Facial Recognition:** Uses `face-api.js` to identify users in real-time.
* **Autonomous Memory:** Automatically records and updates conversation history using Web Speech API.
* **NLP Extraction:** Uses the `natural` library to extract topics and tags from speech.
* **Zero-Login UX:** Recognition is based entirely on facial descriptors.

### Tech Stack
* **Frontend:** React, Tailwind CSS, face-api.js
* **Backend:** Node.js, Express, Mongoose, Natural NLP
* **Database:** MongoDB
